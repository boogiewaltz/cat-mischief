# Cat Mischief - Three.js Prototype

A playful 3D cat sandbox game where you cause mischief around a house using paw swipes.

## Controls

- **WASD**: Move
- **Space**: Jump
- **Shift**: Sprint
- **Q**: Left paw swipe
- **E**: Right paw swipe
- **F3**: Toggle debug info

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm run preview
```

## Features (MVP)

- Third-person cat movement with physics
- Left/right paw swipe interactions
- Knockable objects
- Scratchable surfaces
- Task system with progress tracking
- Mischief scoring

## Tech Stack

- Three.js (3D rendering)
- TypeScript
- Vite (build tool)
- @dimforge/rapier3d (physics - optional)
